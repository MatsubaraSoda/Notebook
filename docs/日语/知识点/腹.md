# 腹

**体言**

**振り仮名**：はら

**同音異字**：腹・胎・肚（胎・肚，这两个字几乎不用了。）

## 意味

<div class="vocab-term">
<div class="vocab-term-title">肚子</div>
<div class="vocab-term-content">
既指与背部相对的肚子，也指肠胃。

"肚子饿了"有两种常见表达：「<ruby>腹<rt>はら</rt></ruby>が減った」「お<ruby>腹<rt>なか</rt></ruby>が空いた」。注意其中「腹」字的读音是不同的。

<ruby>腹<rt>なか</rt></ruby>似乎只有肚子饿了这一种用法。
</div>
</div>

<div class="vocab-term">
<div class="vocab-term-title">心情</div>
<div class="vocab-term-content">
「腹が立つ」和「腹を立てる」是 「腹」最常出现的词组。

为什么是「腹が立つ」而不是 \*「腹に立つ」。我的理解是（不保真）这个词组是用来形容心情的，所以用「が」,类似的有：「～が好き」「～が怖い」「～が気になる」。
</div>
</div>

<div class="vocab-term">
<div class="vocab-term-title">胎内</div>
<div class="vocab-term-content">
和中文的表达相似："肚子大了"、"肚子里有小孩"。
</div>
</div>

## 补充

### 「役に立つ」と「役に立てる」

在语料库找到「腹」最常出现的词组是「腹が立つ」和「腹を立てる」之后，便试着在语料库中搜索「立つ」和「立てる」最常见的词组。分别是「役に立つ」和「役に立てる」。

两者的关系见：[HiNative](https://ja.hinative.com/questions/19464987)

> 「役に立てる」 = 「役に立つことができる」 \
> 「あなたの役に立てて嬉しいです。」は「あなたの役に立つことができて嬉しいです。」という意味です
