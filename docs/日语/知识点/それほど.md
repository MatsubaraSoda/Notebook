# それほど

<div class="vocab-term">
<div class="vocab-term-title">①「それほど」＝「そんなに」＝「そう」</div>
<div class="vocab-term-content">
物事の状態が思ったほどでないことを表す。
<br>
“不很……”。并非那么……，并不……。
</div>
</div>

<div class="vocab-term">
<div class="vocab-term-title">②「それほど」＝「そんなに」＝「そう」</div>
<div class="vocab-term-content">
物事の程度がはなはだしいことを表す。
<br>
那么，那样。表示事情的程度过甚。
</div>
</div>
