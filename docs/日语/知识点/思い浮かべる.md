# 思い浮かべる

<div class="vocab-term">
<div class="vocab-term-title">①</div>
<div class="vocab-term-content">
心の中に描く。思い出す。想起する。
</div>
</div>

<div class="vocab-term">
<div class="vocab-term-title">②</div>
<div class="vocab-term-content">
頭に描く。連想する。
</div>
</div>

## 「思い～」

带有「思い～」的复合动词有：「思い出す」「思いつく」「思い込む」「思い切る」
